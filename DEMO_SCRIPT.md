# 1 分钟演示视频脚本

目标：录一段 60-75 秒视频，用于 GitHub 项目说明或项目汇报。视频只需要展示“项目能跑、结构清楚、有规模、有价值”。

## 录制顺序

1. 打开首页，停留在项目标题和左侧指标。
2. 鼠标划过指标，口头说明十万级论文和百万级关系。
3. 在图谱中搜索 `graph` 或选择 `AAAI` / `ACL`。
4. 点击一个论文节点，展示右侧详情和邻接关系。
5. 往下滚动到 Pipeline，说明完整流程。
6. 往下滚动到 Schema 和 Evidence，展示项目产出。
7. 结尾回到未来扩展区域。

## 口播稿

这个项目是我在课题组中独立完成的论文知识图谱与科研知识底座。它的目标是把原始论文元数据组织成可查询、可分析、可持续更新的图结构。

项目流程包括论文 JSON 解析、字段清洗、实体归一化、稳定 ID 去重、Neo4j 图数据库导入和可视化展示。最终在本地完成了十万级论文数据和百万级关系的入库验证。

这里展示的是一个脱敏抽样的 Web Demo，可以按会议、关键词和节点类型筛选，也可以点击节点查看论文详情和邻接关系。完整图谱仍然存储在 Neo4j 中，展示页主要用于快速说明项目形态。

后续这个知识底座可以继续接入引用数据、语义检索和科研问答模块，为课题组的文献分析和研究脉络挖掘提供基础支撑。

## 录屏建议

- 分辨率用 1920x1080。
- 浏览器缩放保持 100%。
- 不要暴露本地路径、隐私文件或课题组原始数据。
- 视频文件名建议：`paper_kg_showcase_demo.mp4`。
- 上传 B 站、GitHub Release 或放到个人主页都可以。
